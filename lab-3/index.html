Task 2: CSS Selector Specificity and Priority

Span Element Color Color When Mouse Over Explanation
span.myclass1 pink blue Normal: #header span (specificity: 101) matches, overrides span (1). On hover: #header span:first-child:hover (specificity: 103) applies.
span.myclass2 pink pink #header span (specificity: 101) matches. No hover rule applies to this element.
span.myclass3 orange orange This is the last child inside #header, so #header *:last-child (specificity: 102) applies and is more specific than #header span (101).
span.myclass4 red red This span is inside <p>, not #header, so p > span (specificity: 2) matches. It's not affected by #header rules.
